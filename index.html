Thirukkural App Support

This app displays all 1330 Thirukkural verses with classical Tamil commentaries.

For support or feedback, email: jayakanthan@me.com

Features

  • NavigationSplitView — 3-column layout on iPad, stack on iPhone
  • 7 Tamil commentaries + English translations
  • Full-text search in Tamil and English
  • Share any kural via the share sheet
  • Offline access after first data load (cached in Documents)
  • Dark/Light mode follows system setting
  • Keyboard navigation on iPad with arrow keys
